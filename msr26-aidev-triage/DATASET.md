# Dataset Instructions for AIDev

> [!IMPORTANT]
> This replication package requires the **AIDev dataset v1.0 (October 2025 snapshot)**.
> The dataset is NOT included in this package due to size constraints.

**Official Source**: https://huggingface.co/datasets/hao-li/AIDev

### Quick Download (Recommended)

```bash
# Install Hugging Face CLI
pip install huggingface-hub

# Create data directory
mkdir -p data/raw

# Download dataset
huggingface-cli download hao-li/AIDev --repo-type dataset --local-dir data/raw/
```

### Manual Download

1. Visit https://huggingface.co/datasets/hao-li/AIDev
2. Download these files:
   - `pull_requests.parquet` (~2GB)
   - `pr_commits.parquet` (~1.5GB)
   - `pr_comments.parquet` (~500MB)
3. Place in `data/raw/` directory
